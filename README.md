# UTN-TPIOE
Proyecto Integrador de la materia Organización Empresarial (Tecnicatura Universitaria en Programación - UTN).
Simulador de chatbot para la gestión de solicitudes de vacaciones, diseñado para optimizar el proceso administrativo de Recursos Humanos mediante la validación automática de saldos y la gestión de estados.
Python 3.x instalado.
1. Clonar el repositorio: `git clone https://github.com/larabellingi-glitch/UTN-TPIOE.git`
2. Ejecutar el script principal: `python TPI_OE.py`
